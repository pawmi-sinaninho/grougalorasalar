# NEW CHAT WORKFLOW

Use one chat per specification phase unless the phase explicitly involves implementation.

## Attach to each new chat

1. latest cumulative master MD;
2. latest complete phase ZIP;
3. any new screenshots, recordings or observations.

## Required opening check

The assistant should state:

1. files found;
2. exact phase deliverable;
3. genuinely missing evidence, if any.

## Required final output

- updated cumulative master MD;
- complete phase ZIP;
- unresolved issue list;
- exact next-phase brief.

## Codex restriction

Do not create or use a Codex execution prompt before the Phase-8 entry gate in `docs/product/PRE_LIVE_DELIVERY_PLAN.md` is satisfied.
